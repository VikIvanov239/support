git init                                - запускаем cmd в нужной папке  (создаем невидимого друга) создаем репозиторий(проект)
git add .                               - подготовка файлов для фиксации (подготовка к броску) add .(все файлы)
git commit -m "описание изменений"      - фиксирем файлы (готовы к броску)

git remote add origin https://github.com/VikIvanov239/html_css.git  подготовка к push(только для первого push) интуитивно понятно на github(создание на сайте репозитория)
git push                                - заливаем на аккаунт в github(первый пуш сложный)
git status                              - проверяем состояние репозитория (в любой непонятной ситуации)
git checkout .                          - вернули в состояние последней фиксации(commit-а).

git clone git@github.com:VikIvanov239/support.git     скачиваем репозиторий на другой компьютер.
git pull                                         достаем  новые фиксации из облака на на данный компьютер.
